Demo Box Model CSS

Perbandingan box-sizing: content-box vs box-sizing: border-box

content-box (default)

Dimensi Total:
Lebar konten: 200px
+ padding (kiri+kanan): 40px
+ border (kiri+kanan): 10px
= 250px total lebar elemen
➕ Margin (kiri+kanan): 40px

border-box

Dimensi Total:
Lebar total elemen: 200px
(padding dan border sudah termasuk)
➕ Margin (kiri+kanan): 40px

Lebar konten tersisa:
200 − 40(padding) − 10(border) = 150px

Kesimpulan

Kedua kotak di atas memiliki properti yang sama: width: 200px; padding: 20px; border: 5px solid; margin: 20px;. Perbedaannya hanya pada box-sizing.

content-box — lebar 200px hanya untuk konten. Padding dan border ditambahkan di luar sehingga total lebar elemen menjadi 250px.

border-box — lebar 200px sudah mencakup konten, padding, dan border. Sehingga ukuran elemen tetap 200px dan lebih mudah diprediksi.
